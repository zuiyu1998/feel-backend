## 概述

用户展示的第一印象。用户给自己贴的标签，可以缩小用户之间的认知负担，寻找同好，避免无效交流。

## 指南级别的解释

用户标签有一个名称和一个简单描述。点击标签可以发现标签的生成时间和生成原因等数据。标签模块需要实现的功能。

- 创建标签模板
- 用户创建标签
- 用户的第一印象

## 参考级别解释

标签相关对象的声明如下:

```proto
//获取用户印象的参数
///获取用户印象的参数
message UserLabelParams {
    int32 id = 1;
    int32 page_size = 2;
    int32 page = 3;
}

///分页获取标签的响应
message UserLabelResponse {
    int32 count = 1;
    int32 page_size = 2;
    int32 page = 3;
    repeated UserLabel data = 4;
}

///用户创建标签的参数
message UserLabelCreate {
    int32 user_id = 1;
    int32 label_meta_id = 2;
}

///用户创建标签成功的响应
message UserLabelCreateResponse {
    int32 id = 1;
}

///标签模板创建的参数
message UserLabelMetaCreate {
    /// 名称
    string name = 2;
    /// 描述
    string description = 3;
    ///影响力
    int64 effct = 4;
}

///标签模板创建成功的响应
message UserLabelMetaCreateResponse  {
    int32 id = 1;
}

///标签
message UserLabel {
    /// id
    int32 id = 1;
    UserLabelMeta meta = 2;
    int64 create_at = 3;
    int64 update_at = 4;
}

//标签模板
message UserLabelMeta {
    /// id
    int32 id = 1;
    /// 名称
    string name = 2;
    /// 描述
    string description = 3;
    ///影响力
    int64 effct = 4;
    int64 create_at = 5;
    int64 update_at = 6;
}

```
