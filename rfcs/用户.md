## 概述

社区中最基础也最重要的单元。

## 指南级别的解释

每一个实际的用户都拥有社区的一个抽象。用户的定义如下：

- 昵称(唯一)
- 头像
- uid
- 创建时间
- 更新时间
- id (数据库 id)

用户的实际功能如下:

- 注册
- 注销
- 登录
- 用户信息
- 更改用户数据

## 参考级别解释

用户基础声明如下:

```proto
//用户基础数据
message UserBase {
    int32 id = 1;
    string nikename = 2;
    string avatar = 3;
    string uid = 4;
    int64 create_at = 5;
    int64 update_at = 6;
}

//用户授权数据
message UserAuth {
    enum AuthType {
        Email = 0;
    }

    int32 id = 1;
    AuthType auth_type = 2;
    string auth_name = 3;
    bytes auth_code = 4;
}

//用户注册
message UserRegister {
    string nikename = 1;
    string avatar = 2;
    string uid = 3;
    AuthType auth_type = 4;
    string auth_name = 5;
    bytes auth_code = 6;
}

//用户信息
message UserInfo {
    int32 id = 1;
    string nikename = 2;
    string avatar = 3;
    string uid = 4;
    int64 create_at = 5;
    int64 update_at = 6;
}

//用户更新
message UserUpdate {
    int32 id = 1;
    optional string nikename = 2;
    optional string avatar = 3;
}

message UserUnregister {
    int32 id = 1;
}

message GetUserInfoParams {
    int32 id = 1;
}


```

用户 api 功能声明如下:

```rust
//注册
async fn register(register: UserRegister) -> UserBase;
//注销
async fn unregister(user_id: i32) -> UserBase;

//登录
async fn login(login: UserLogin) -> UserBase;

//用户信息
async fn getUserInfo(user_id: i32) -> UserInfo;

async fn update(update: UserUpdate) -> UserBase;
```

grpc DbService 服务声明如下:

```proto
service DbService {

  rpc Register(UserRegister) returns (UserBase);
  rpc Unregister(UserUnregister) returns (UserBase);
  rpc Login(UserLogin) returns (UserBase);
  rpc GetUserInfo(GetUserInfoParams) returns (UserBase);
  rpc Update(UserUpdate) returns (UserBase);

}
```
